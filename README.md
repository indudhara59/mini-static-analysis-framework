# Mini Static Analysis Framework

A small Java framework for experimenting with classical static analysis algorithms.

## Project Status

Work in progress.

## Goal

The goal of this project is to implement basic static analysis concepts such as intermediate representations and dataflow analyses.

## Technologies

- Java
- Maven
- Dataflow Analysis

## Planned Features

- Design a simple intermediate representation
- Implement Reaching Definitions analysis
- Implement Live Variable Analysis
- Add example programs for analysis
- Document analysis results and behavior

## Current Focus

Initial framework design and basic analysis structure.
